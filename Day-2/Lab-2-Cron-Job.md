# Lab 2 – Cron Job Automation

## Objective
Automate a task using cron scheduling.

## Tasks Performed
1. Opened cron configuration using `crontab -e`.
2. Added a cron job to run every 5 minutes.
3. Logged output to `/tmp/test.log`.

## Result
Cron job successfully executed and updated the log file automatically.
