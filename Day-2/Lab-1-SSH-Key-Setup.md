# Lab 1 – SSH Key Setup

## Objective
Configure secure SSH login using key-based authentication.

## Tasks Performed
1. Generated SSH key pair.
2. Copied the public key to the remote server.
3. Disabled password authentication for SSH.
4. Verified login using SSH keys.

## Result
Successfully configured password-less and secure SSH access.
